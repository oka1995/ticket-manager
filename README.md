# ポケモンカード管理アプリ

法人名義で古物商許可を取得した業者向けの、ポケモンカード売買管理アプリ。
Google スプレッドシートをデータベースとして、古物台帳・在庫・売上・税務記録を一元管理する。

## 主な機能

- **古物台帳**（法定記録）: 買受・売却ごとに自動記帳。3年間の保管義務に対応
- **在庫管理**: 仕入カードの状態・保管場所・販売状況を追跡
- **売上台帳**: 販路別の売上・手数料・送料・粗利を記録
- **税務レポート**: 月次/年次の売上・原価・粗利の集計と CSV 出力
- **Claude Vision によるカード自動認識**: 写真をアップロードするとカード名・拡張パック・レアリティを自動入力
- **ダッシュボード**: 在庫評価額・当月売上・粗利・過去6ヶ月推移をグラフ表示

## セットアップ

### 1. Google スプレッドシートの準備

1. Google ドライブで新規スプレッドシートを作成（名前は任意、例: `ポケモンカード管理DB`）
2. 「拡張機能」→「Apps Script」を開く
3. `Code.gs` の内容をコピー＆ペーストして保存
4. （任意）プロジェクトの設定 → スクリプトプロパティ → `ANTHROPIC_API_KEY` に Anthropic API Key を設定（写真解析を使う場合）

### 2. ウェブアプリとしてデプロイ

1. Apps Script エディタで「デプロイ」→「新しいデプロイ」
2. 種類: **ウェブアプリ**
3. 実行ユーザー: **自分**
4. アクセス権: **全員**
5. デプロイ完了後に表示される URL をコピー

### 3. フロントエンド

1. `index.html` をブラウザで開く（ローカル or 静的ホスティング）
2. 上部のバーに GAS URL を貼り付け、「接続＆初期化」をクリック
3. シートが自動作成される

## データ構造

### 古物台帳（法定記録 / 3年保管義務）

古物営業法第16条に基づく必須項目を含む:
- 取引年月日、取引区分（買受/売却）
- 品目（カード名・拡張パック・言語・レアリティ・状態）、数量、代価
- 相手方の氏名・住所・職業・年齢・確認方法
  - 買受 1万円以上、または美術品類・交換は必須
  - オンライン非対面取引は一部簡略化可（詳細は自治体の公安委員会指示に従う）

### 在庫台帳

仕入れ済みカードの実在庫管理。売却時にステータスが自動更新される。

### 売上台帳

販路（メルカリ・ヤフオク・TCGショップ等）ごとの売上と、手数料・送料・粗利を自動計算。

## 法的注意事項

- 本アプリは古物営業法 第16条（帳簿等への記載）に対応する項目を備えていますが、**最終的な法令遵守の責任は運営者にあります**。所轄警察署（生活安全課）の指導内容を優先してください。
- 法人決算では別途、会計ソフトへの仕訳入力が必要です。CSV 出力機能を活用して弥生会計等へインポートしてください。
- 消費税の適格請求書（インボイス）対応は本アプリの範囲外です。
